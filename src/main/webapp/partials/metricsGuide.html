Taiga-Github-Slack Frequency


Taiga

Refers to the amount of days in the week that task state changes were made, aggregated to a proportional number. Averages for total Course and Team, individual totals for Student


GitHub

Refers to the amount of days in the week that commit activity was done, aggregated to a proportional number. Averages for total Course and Team, individual totals for Student


Slack

Refers to the amount of days in the week that messages were made in the group, aggregated to a proportional number. Averages for total Course and Team, individual totals for Student


Taiga-Github-Slack Impact(Weight)


Taiga

Refers to the amount of task state changes that were made for a week, aggregated to a proportional number. Averages for total Course and Team, individual totals for Student


GitHub

Refers to the amount of commits that were made for a week, aggregated to a proportional number. Averages for total Course and Team, individual totals for Student


Slack

Refers to the amount of messages that were sent in a group for a week, aggregated to a proportional number. Averages for total Course and Team, individual totals for Student


Taiga Data


Task Totals


Refers to the individual task state amounts per day collected: 'In Progress', 'Ready For Test', and 'Closed' These are selectable by week/multi-week ranges. Averages for total Course and Team, individual totals for Student


Task Status Update Activity


Refers to the daily difference/change in task states: 'In Progress', 'Ready For Test', and 'Closed' This is listed by week, from aggregated totals. Averages for total Course and Team, individual totals for Student

GitHub Data


Note: GitHub API supplies commit and LOC data by week beginning only, this is why it is listed as such in the graphs


GitHub Commit Activity


Commits

Refers to the number of commits per week, that are made to the master branch. Other branch commits are factored in upon pull request merge, is selectable by week/multi-week ranges. Averages for total Course and Team, individual totals for Student.


Lines of Code

Refers to the Lines of Code Added and also Deleted per week, that are made to the master branch, Other branch lines of code are factored in upon pull request merge, is selectable by week/multi-week ranges. Averages for total Course and Team, individual totals for Student



GitHub Weight Activity


Refers to an aggregated proportional value calculated from the total code alteration, ie Lines of Code Added and also Deleted per week, that are made to the master branch, other branch lines of code amounts are factored in upon pull request merge. Averages for total Course and Team, individual totals for Student

Slack Data


Message Totals


Refers to the daily slack message totals for messages sent in a group, is selectable by week/multi-week ranges. Averages for total Course and Team, individual totals for Student


Message Activity


Refers to the weekly slack message totals for messages sent in a group. Averages for total Course and Team, individual totals for Student
